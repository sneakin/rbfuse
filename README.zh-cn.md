# RBfuse

简单的使用 [FFI](https://github.com/ffi/ffi)  实现的 FUSE ruby 库

目前只是一个玩具项目，没有测试，暂时只有 OSX 上可以使用

## Require  

* fuse or osxfuse 2.x
* ruby 2.x

## Installation


```ruby
gem 'rbfuse'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rbfuse
