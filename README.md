# RBfuse

Ruby FUSE, with [FFI](https://github.com/ffi/ffi)

Support OSX And Linux 64 bit system

## Require  

* fuse or osxfuse 2.x
* ruby 2.x

## Installation


```ruby
gem 'rbfuse'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rbfuse
