# Rubyfuse

FUSE with Ruby

NOTE: This is just one of my toys project

## Require  

* fuse or osxfuse 2.x
* ruby 2.x

## Installation


```ruby
gem 'rbfuse'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rbfuse
